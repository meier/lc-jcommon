This work was produced at the Lawrence Livermore National Laboratory under Contract No. DE-AC52-07NA27344 with the DOE. ------------------------------------------------------------------------------ This Package contains a simple logging mechanism. To use it from within any class, simply tag the class as implementing the gov.llnl.lc.logging.ComonLogger interface. For example; 
package gov.llnl.lc.meier3.opensm.test;

import gov.llnl.lc.logging.CommonLogger;

public class LoggerExample implements CommonLogger
{

  public static void main(String[] args)
  {
    logger.info("this is only a test");
  }

}
Produces the following output (using defaults) on the console; 
Aug 26, 2011 3:25:01 PM gov.llnl.lc.meier3.opensm.test.LoggerExample main
INFO: this is only a test
